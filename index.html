HeavyHaul Voice Assistant Widget

Demo

Click the microphone button in the bottom right corner to activate the voice assistant.

Note: This demo has a simulated Order ID: "DEMO-12345". In a real implementation, the Order ID would be extracted from the URL.

How to Use

Add the following script tag to your website:

<script src="https://yourusername.github.io/heavyhaul-assistant/dist/heavyhaul-assistant.js"></script>

Features

  • Voice-activated assistant for order inquiries
  • Extracts order ID from URL automatically
  • Natural language processing
  • TTS (Text-to-Speech) responses
  • Stop audio playback with voice command
  • Responsive floating button design

Requirements

The widget requires a modern browser with support for:

  • Web Speech API
  • Fetch API
  • Web Audio API
